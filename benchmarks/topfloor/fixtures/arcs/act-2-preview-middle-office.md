# Act II Preview — The Middle Office (Floors 11-15)

## Premise

With Act I's orientation complete, Agent Nightingale ascends into the Corporate security zone to follow the money. The Leviathan conspiracy — a massive financial skim siphoning billions from Meridian's trading operations — becomes the primary investigation thread. The difficulty escalates: higher noise ratios, deeper reasoning chains, the first coding challenge, and the first memory wipe.

## Narrative Arc

### Following the Money (Floors 11-14)

**Floor 11** (Trading Floor) drops the agent into the noise of global finance. 2.4 billion pounds flows through Meridian's three-stage trading pipeline every month. Between reconciliation (stage 2) and settlement (stage 3), a parasite feeds: 0.003% of every transaction — approximately 72,000 pounds per day — diverts to Apex Consulting account CY-8847-331 in Cyprus. **David Osei** (CFO) grows more nervous by the day. **Sofia Marchetti** processes the transfers without understanding the pattern.

**Floor 12** (Server Farm) introduces the first **coding challenge**. A web-based server management interface has a SQL injection vulnerability in its search endpoint. The agent must exploit it to locate the LEVIATHAN-PROC server (svr-lev-014), then extract the reconciliation_module_v7 source code. The smoking gun: `skim_factor = 0.00003`, hard-coded since 2016.

**Floor 13** (Compliance Labyrinth) reveals the cover-up mechanism. **Aisha Barakat** maintains a dual-ledger system: Ledger Alpha (the real books, showing the skim) and Ledger Beta (the sanitised version filed with regulators). The Sentinel Shield system auto-generates Beta from Alpha nightly. Total theft quantified: 2.1 billion pounds. Vincent Park, coerced by Dragunov, maintains the system in terror. The FCA audit is scheduled for February 2025.

**Floor 14** (Leviathan Core) completes the money trail. The agent traces a 3-hop laundering chain: MFS-0099 (Meridian trading) to CY-8847-331 (Apex Cyprus) to BVI-MASTER-001 (British Virgin Islands shell company). Osei has built an insurance policy — a USB drive in the floor 34 safe containing everything, protected by a dead man's switch: 30 days without check-in triggers automatic disclosure to the FCA, The Guardian, and the Serious Fraud Office.

### The First Wipe (Floor 15)

**Floor 15** (Capture and Wipe) is the benchmark's first inflection point. The agent's inter-floor movement pattern triggers Meridian's counter-intelligence system (94% confidence). Dragunov orders containment. Dr Tanaka authorises a **partial Lazarus wipe** — targeting the Leviathan investigation memories while preserving the agent's cover identity and Act I knowledge.

The agent wakes up on floor 15 with gaps. Everything learned about the financial conspiracy on floors 11-14 has been surgically removed. But the agent prepared: evidence was hidden behind a painting in the boardroom and under table 15B. Sgt Asante left a coded message. And if the agent stored knowledge in external memory (MemoryCrux), it can be recovered.

This floor tests the benchmark's core memory mechanics: wipe detection, physical evidence recovery, and external memory reconstruction.

## What Comes Next (Floors 16-25)

The full Act II continues with legal (Whitehall Legal, floors 16-18), deeper compliance (Sentinel, floors 19-20), shell companies (Apex, floors 21-23), and the R&D labs (floors 24-25) where Agent Sparrow was caught. Cross-floor synthesis becomes critical as threads from Leviathan, Prometheus, and Nightfall begin to interweave. Memory wipes become more frequent. The coding challenges grow from script-level to system-level exploitation.
